Fall 2018 Updates for You Tube and Spring Data Rest

This version of the website pulls You Tube Channel and Playlist data directly from You Tube.

Also, it has been retrofitted to use Spring Data REST on the Events features, and Spring REST controllers on the Performance and Class Schedule Planning features.

General Comments

The website is written in Groovy and Angular6; it is structured as a single page application.

The site runs in SpringBoot and is deployed on Amazon AWS, running within a Docker Container

You can find the source for this site at this Github link

Technologies Employed

  • Angular6
  • Bootstrap
  • Amazon Web Services
  • Spring Boot
  • Spring Data Rest
  • Docker
  • Github
  • Groovy
